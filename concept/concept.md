CAPSTONE PROJECT MANAGEMENT SYSTEM

# This project can be likened to a projects management system. The purpose of this project is primarily to streamline the capstone process between instructors, students, adviser, panelist, dean and panil chair.

# Objectives:
    Specifically the system aims to develop the following:
    1. User Authentication and Access Control Module
    1.1.Login and logout functionality for students, advisers, instructors, panel members,
    and the dean
    1.2.Role-based dashboard display for each system user
    1.3.Permission control for document upload, review, approval, and scheduling
    2. Capstone Group and Role Management Module
    2.1.Create and manage capstone project groups
    2.2.Assign student roles (PM/Analyst, Programmer, Documentarian) within a group
    2.3.Enforce group-level document access and data isolation
    3. Capstone Title Repository Module
    3.1.View existing and approved capstone project titles
    3.2.Search and filter capstone titles for originality checking
    3.3.Prevent duplicate title submission through system validation
    4. Concept Submission and Adviser Review Module
    4.1.Submit three (3) project concepts during Capstone 1
    4.2.Route submitted concepts to the assigned adviser
    4.3.Approve, reject, or request resubmission of project concepts
    4.4.Update concept status and notify students of adviser feedback
    5. Defense Scheduling and Payment Validation Module
    5.1.Assign panel members to capstone project groups
    5.2.Set defense schedules including date, time, and room assignment
    5.3.Validate payment status prior to defense scheduling
    5.4.Restrict defense access for unpaid groups
    6. Document Routing and Panel Evaluation Module
    6.1.Route approved capstone documents to panel members
    6.2.Input panel comments and evaluation scores online
    6.3.Store evaluation results and panel feedback
    7. Verdict Decision and Approval Module
    7.1.Consolidate panel evaluation results
    7.2.Record final verdict as Approved or Re-Defense
    7.3.Display verdict results to authorized users
    8. Automated Minutes and Approval Sheet Module
    8.1.Generate official Minutes of the Meeting
    8.2.Generate PDF Approval Sheet with complete project details
    8.3.Attach digital signatures of the adviser, panelist, and dean
    9. Deadline Monitoring and Notification Module
    9.1.Track capstone deadlines set by the instructor
    9.2.Send dashboard notifications three (3) days before deadlines
    9.3.Allow deadline modification and extension by authorized personnel
    10.Deployment Verification and Archiving Module
